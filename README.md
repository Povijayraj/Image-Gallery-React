# 📸 React Image Gallery

A modern and responsive Image Gallery application built using **React.js** and **Tailwind CSS**. This project showcases a clean, user-friendly interface for displaying images in a responsive grid layout that adapts seamlessly to different screen sizes.

## 🚀 Features

- 📱 Fully responsive design
- ⚛️ Built with React.js components
- 🎨 Styled using Tailwind CSS
- 🖼️ Responsive image grid layout
- ⚡ Fast and lightweight UI
- 💻 Clean and reusable code structure

## 🛠️ Technologies Used

- React.js
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3

## 📂 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/image-gallery.git
   ```

2. Navigate to the project folder:
   ```bash
   cd image-gallery
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## 🎯 Purpose

This project was created to practice building responsive user interfaces with React.js and Tailwind CSS while improving component-based development and modern frontend styling techniques.
